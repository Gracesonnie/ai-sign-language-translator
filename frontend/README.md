# AI Sign Language Translator - Frontend

This is the **Angular frontend** for the AI Sign Language Translator application.

---

## 📋 About This Project

This is a **customer service gesture assistant** that uses AI to recognize sign language gestures. It detects **Hello** and **Stand** gestures using a Teachable Machine model.

---

## 🚀 Technologies Used

| Technology            | Purpose                   |
| --------------------- | ------------------------- |
| **Angular 22**        | Frontend framework        |
| **TypeScript**        | Main programming language |
| **TensorFlow.js**     | AI model runtime          |
| **Teachable Machine** | Gesture recognition model |

---
